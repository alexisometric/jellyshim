Cache Headers

Add Cache-Control, ETag, and Vary headers to static files.
Cache duration for hashed/fingerprinted assets (e.g., main.a1b2c3d4.js). Default: 31536000 (1 year, immutable).
Cache duration for general static assets. Default: 2592000 (30 days).
Cache duration for community plugin static assets. Default: 86400 (1 day).
Serve stale content while revalidating in background. Default: 86400 (1 day).

Image Optimization

✓ Built-in image processing

JellyShim processes images natively using ImageSharp + SkiaSharp — no external service required.
Images are resized, re-encoded (AVIF/WebP/JPEG), and cached to disk automatically. Each Jellyfin image type has independent width and quality settings.

Intercept image requests and process them with resizing, quality reduction, and format conversion.
Cache processed images to disk for instant serving on subsequent requests.
Preferred output format: avif, webp, jpeg, or auto (auto negotiates AVIF → WebP → JPEG based on browser support).
Cache-Control max-age for image responses. Default: 2592000 (30 days).
Serve stale image while revalidating in background. Default: 604800 (7 days).
